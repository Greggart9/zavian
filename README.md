# Zavian

Zavian is a visually driven web project built to explore and practice advanced frontend animations, smooth scrolling interactions, and modern motion-based UI experiences.

The main goal of this project is to improve my animation skills by creating a polished, interactive, and immersive portfolio-style website experience.

## Overview

Zavian focuses on combining clean visual design with smooth motion. The project includes animated sections, image-based storytelling, hover interactions, scroll-triggered effects, and refined transitions to create a premium browsing experience.

Rather than being a static website, Zavian is designed as a practice project for experimenting with how motion can improve user engagement, guide attention, and make a digital experience feel more dynamic.

## Purpose of the Project

The primary purpose of building Zavian was to practice and improve my frontend animation skills.

This project allowed me to explore:

- Scroll-based animations
- Image zoom and reveal effects
- Smooth page transitions
- Interactive hover states
- Layered visual compositions
- Motion timing and easing
- Responsive animation behavior
- Building a more polished visual experience with code

## Features

- Modern landing page layout
- Full-screen featured project sections
- Scroll-triggered image animations
- Interactive thumbnail hover effects
- Smooth scrolling experience
- Custom cursor interaction
- Responsive visual layout
- Clean typography and spacing
- Reusable component structure

## Tech Stack

- **Next.js** — React framework for building the application
- **TypeScript** — For type safety and better development workflow
- **Tailwind CSS** — For styling and responsive layouts
- **GSAP** — For animation and scroll-triggered motion effects
- **Next/Image** — For optimized image rendering

## What I Practiced

This project helped me practice important animation and frontend concepts such as:

- Creating smooth scroll-based effects with GSAP
- Using animation timing to make interactions feel natural
- Structuring reusable UI components
- Managing image layers, overlays, and z-index behavior
- Improving visual hierarchy through motion
- Building immersive project showcase sections
- Combining design thinking with frontend implementation
